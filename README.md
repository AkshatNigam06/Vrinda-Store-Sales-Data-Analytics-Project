# Vrinda-Store-Sales-Data-Analytics-Project
Interactive Excel dashboard built using Pivot Tables, Charts, and Slicers to generate business insights from sales data.


**Key Insights**
Women contributed 64% of total sales.
Amazon generated 35% of total orders.
Maharashtra was the highest-selling state.
March recorded the highest sales performance.
Delivered orders accounted for 92% of all orders.
 
**Key Features**
📊 Built an interactive Excel dashboard for sales analysis.
📈 Analyzed monthly sales and order trends.
👩‍💼 Examined customer demographics by age and gender.
🛒 Evaluated order distribution across multiple sales channels.
📦 Tracked order status including delivered, returned, and refunded orders.
🌍 Identified top-performing states based on sales revenue.
🎯 Used slicers for dynamic filtering by month, channel, and category.
📉 Generated business insights to support data-driven decision-making.
